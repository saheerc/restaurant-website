# La Bella – Restaurant Website

A fully responsive Italian restaurant website with a filterable menu, reservation form, and contact section. Built with HTML, CSS, and vanilla JavaScript.

## Features

- Full-screen hero with real background image
- About section with chef story
- Filterable menu (All / Starters / Mains / Desserts)
- Table reservation form with date, time, and guest count
- Contact info and location section
- Mobile responsive layout
- Sticky navigation

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables)
- Vanilla JavaScript (menu filtering)

## Getting Started

```bash
git clone https://github.com/saheerc/restaurant-website.git
cd restaurant-website
open index.html
```
